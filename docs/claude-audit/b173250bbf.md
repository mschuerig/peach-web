## Commit message

Code review fixes for story 8.5: SF2 always-fetch, preset header parsing, settings UI polish

- Always fetch SF2 at app startup (remove oscillator-only guard that prevented SF2 discovery)
- Parse SF2 preset headers from raw RIFF/sfbk bytes at fetch time (no AudioContext needed)
- Settings Sound row shows loading indicator while SF2 fetches, then full preset dropdown
- Move hint text outside SettingsSection cards to match iOS footer pattern
- Add chevron indicators on Sound and Tuning dropdown selects
- Extract duplicated interval toggle button classes to TOGGLE_ACTIVE/TOGGLE_INACTIVE constants
- Use .round() as i32 for concert pitch display instead of truncation
- Remove dead SoundFontLoadStatus::NotNeeded variant and unused import

Co-Authored-By: Claude Opus 4.6 <noreply@anthropic.com>

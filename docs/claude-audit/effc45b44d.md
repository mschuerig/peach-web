## Story 8.10: Settings Sound Source Preview

**Story ID:** 8.10
**Story Key:** 8-10-settings-sound-source-preview

**As a** user,
**I want** to preview the selected sound source from the Settings screen,
**so that** I can hear what a sound sounds like before starting a training session.

### Acceptance Criteria

1. Given the Settings screen is showing, when the user looks at the Sound section, then a `speaker.wave.2` SF Symbol-equivalent icon button is visible to the right of the Sound picker
2. Given no preview is playing, when the user taps the preview button, then a 2-second note plays using the currently selected sound source at the current concert pitch (A4)
3. Given a preview is playing, when the user taps the preview button again, then the playing sound stops immediately
4. Given a preview is playing, when the user changes the sound source in the picker, then the playing preview stops
5. Given a preview is playing, when the user navigates away from Settings, then the playing preview stops
6. Given the preview button is tapped and the note finishes its 2-second duration naturally, then the button returns to its default icon (not stuck in stop state)
7. Given the AudioContext has not been created yet (no user gesture), when the user taps preview, then the AudioContext is created (the tap counts as a user gesture) and the preview plays
